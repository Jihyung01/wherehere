## 📱 전체 기능 목록

### 하단 네비게이션 (5개 탭)
1. **🏠 홈** - 역할 선택 화면
2. **🗺️ 나의 지도** - 방문 기록 (Kakao Map)
3. **🎯 챌린지** - 도전 과제
4. **👤 프로필** - 사용자 프로필
5. **⚙️ 설정** - 앱 설정

### 메인 플로우
```
역할 선택 → 기분 선택 → AI 추천 → 퀘스트 수락
  ↓
소셜 공유 (카카오톡, 인스타, 트위터, 페이스북)
  ↓
체크인 → 리뷰 작성 → XP 획득 → 나의 지도
```

### 설정 기능
- **🌙/☀️ 다크모드 토글** - 라이트/다크 모드 전환
- **🔔 알림 설정** - 푸시 알림 관리
- **🗺️ 위치 서비스** - GPS 권한 관리
- **🔒 개인정보** - 계정 및 보안
- **❓ 도움말** - 사용 가이드

### 소셜 공유
- **💬 카카오톡** - 카카오톡 친구에게 공유
- **📷 인스타그램** - 인스타그램 스토리/피드
- **🐦 트위터** - 트위터에 트윗
- **📘 페이스북** - 페이스북에 게시

### 챌린지 시스템
- **🔥 7일 연속 방문** - 500 XP
- **🗺️ 지역 탐험가** - 300 XP
- **⭐ 별점 마스터** - 200 XP

## 🎨 테마 시스템

### 라이트 모드 (기본)
```css
배경: #FFFFFF (흰색)
텍스트: #1F2937 (진한 회색)
카드: #F9FAFB (연한 회색)
테두리: #E5E7EB (회색)
```

### 다크 모드
```css
배경: #0A0E14 (진한 남색)
텍스트: #FFFFFF (흰색)
카드: rgba(255,255,255,0.05)
테두리: rgba(255,255,255,0.1)
```

### 테마 전환 방법
1. 하단 네비게이션에서 **⚙️ 설정** 클릭
2. 첫 번째 항목 **다크 모드** 토글 클릭
3. 즉시 전체 앱 테마 변경

## 🚀 실행 방법

### 1. 백엔드 시작
```powershell
cd backend
python -m uvicorn main:app --reload --port 8000
```

### 2. 프론트엔드 확인
- 이미 실행 중: http://localhost:3004
- 새로 시작: `cd frontend-app && npm run dev`

### 3. 접속
- **메인 앱**: http://localhost:3004
- **나의 지도**: http://localhost:3004/my-map-real

## 📊 기능별 상세 가이드

### 1. 소셜 공유 사용법

1. 장소 선택 후 "수락" 화면으로 이동
2. **📢 친구에게 공유하기** 섹션 확인
3. 원하는 플랫폼 버튼 클릭:

   - **📘 페이스북**: 페이스북 공유 화면

#### 공유 내용:
```
WhereHere에서 [장소명]를 발견했어요! 🗺️
[링크]
```

#### 챌린지 확인:
1. 하단 네비게이션 **🎯 챌린지** 클릭

3. 진행률 바로 확인

#### 챌린지 완료:
- 조건 달성 시 자동으로 XP 지급

### 3. 프로필 기능

#### 프로필 확인:
1. 하단 네비게이션 **👤 프로필** 클릭
2. 사용자 통계 확인:
   - 총 방문 횟수
   - 획득 XP
   - 탐험 스타일
   - 선호 카테고리

### 4. 설정 기능

#### 다크모드 전환:
1. **⚙️ 설정** 클릭
2. **다크 모드** 토글 클릭
3. 즉시 테마 변경

#### 기타 설정:
- **🔔 알림 설정**: 푸시 알림 ON/OFF
- **🗺️ 위치 서비스**: GPS 권한 관리
- **🔒 개인정보**: 계정 정보 수정
- **❓ 도움말**: 사용 가이드 보기

## 🔧 문제 해결

### CORS 에러 해결
**증상**: "has been blocked by CORS policy"

**해결**:
1. 백엔드 재시작 (위 명령어 사용)
2. 프론트엔드 새로고침 (Ctrl+R)

### 소셜 공유 안 됨
**증상**: 공유 버튼 클릭 시 아무 반응 없음

**해결**:
- 팝업 차단 해제 필요
- 브라우저 설정 → 팝업 허용

### 다크모드 토글 안 보임
**증상**: 설정에서 다크모드 옵션이 없음

**해결**:
1. 페이지 새로고침
2. 하드 리프레시 (Ctrl+Shift+R)
