# CLAUDE.md — Project Instructions

## 파일 수정 규칙

### complete-app.tsx (약 3000줄)
- **전체 덮어쓰기 절대 금지.** 파일이 크기 때문에 Write 도구로 전체를 교체하면 diff 계산과 에디터 로딩에 심각한 랙이 발생한다.
- 반드시 **Edit 도구**로 변경이 필요한 부분만 정확히 타겟해서 수정한다.
- 새 기능 추가 시에도 해당 삽입 위치만 Edit으로 처리한다.

### 예외 — 대규모 개발/향상 요청
- 사용자가 "발전", "향상", "대규모 개선" 등의 표현으로 큰 작업을 요청하거나, Claude 스스로 판단했을 때 대규모 코드 작성이 필요하다고 판단되면 Write로 전체 재작성해도 된다.
- 이 경우 작업 전에 "대규모 수정을 진행합니다" 라고 사용자에게 먼저 알린다.

### 일반 원칙
- 큰 파일(500줄 이상)은 모두 Edit 도구 우선 사용.
- Write 도구는 새 파일 생성 또는 소규모 파일의 완전 재작성에만 사용.
