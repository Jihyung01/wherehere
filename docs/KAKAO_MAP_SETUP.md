# 카카오맵(나의 지도) 도메인 등록

나의 지도 탭은 **카카오맵 JavaScript API**를 사용합니다.  
지도가 "로딩 지연" 또는 "스크립트를 불러오지 못했습니다"로 나오면 **사이트 도메인 미등록**일 가능성이 큽니다.

## 해결 방법

1. **https://developers.kakao.com** 로그인 → **내 애플리케이션**
2. 사용 중인 앱 선택 (지도용 **JavaScript 키**가 있는 앱)
3. **앱 설정** → **플랫폼** → **Web** 플랫폼 추가/수정
4. **사이트 도메인**에 아래처럼 **접속하는 주소**를 등록:
   - 로컬: `http://localhost:3000`
   - Vercel 배포: `https://wherehere-seven.vercel.app` (실제 배포 도메인으로)
5. **저장** 후 브라우저 새로고침

## 참고

- **localhost**와 **배포 도메인**은 서로 다른 도메인이므로, 둘 다 사용할 경우 **둘 다** 등록해야 합니다.
- 지도 스크립트는 `NEXT_PUBLIC_KAKAO_MAP_KEY`(또는 코드 내 기본 앱키)로 로드됩니다. 해당 키가 붙은 앱의 Web 플랫폼에 도메인을 넣으면 됩니다.
