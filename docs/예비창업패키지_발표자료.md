## 📋 Q&A 예상 질문 및 답변

### Q1: 네이버나 카카오가 따라하면 어떻게 하나요?
**A**: 3가지 방어 전략이 있습니다.
1. **특허 출원**: 역할/감정 기반 추천 시스템 (6개월 차)
2. **네트워크 효과**: 친구가 많을수록 가치 증가 (협동 퀘스트)
3. **빠른 혁신**: 월 2회 업데이트로 항상 앞서가기

### Q2: 수익화가 너무 늦지 않나요?
**A**: 의도적인 전략입니다.
- 6개월간 무료로 사용자 확보 (MAU 10,000명)
- 네트워크 효과 형성 후 구독 도입 → 전환율 높음
- 제휴 수익은 3개월 차부터 시작 (월 200만원)

### Q3: AI API 비용이 부담되지 않나요?
**A**: 최적화로 70% 절감했습니다.
- 서사 캐싱: 같은 장소 재사용
- 배치 처리: 여러 요청 한 번에
- 사용자당 월 비용: 약 200원 (MAU 10,000명 기준 월 200만원)

### Q4: 베타 테스터 50명은 너무 적지 않나요?
**A**: 정성적 검증에 집중했습니다.
- 2주간 심층 인터뷰 (각 30분)
- 사용 패턴 분석 (주간 방문 +78%)
- 예비창업패키지 지원 후 200명으로 확대 예정

### Q5: 지역 확장은 어떻게 하나요?
**A**: 3단계 전략입니다.
1. **데이터 수집**: 카카오 API 자동 수집 스크립트 (구현 완료)
2. **사용자 제안**: 크리에이터 모드로 UGC 확보
3. **단계적 확장**: 부산 → 대구 → 광주 (사용자 밀도 순)

### Q6: 경쟁사 대비 기술적 우위는?
**A**: 3가지 핵심 기술입니다.
1. **AI 프롬프트**: 3개월 최적화, 역할/기분별 50가지 조합
2. **PostGIS 공간 쿼리**: 인덱스 최적화로 10배 빠른 검색
3. **게이미피케이션 밸런싱**: 50레벨, 18개 챌린지 정교한 설계

### Q7: 1인 개발의 한계는?
**A**: 예비창업패키지로 해결합니다.
- 개발자 채용 → 개발 속도 2배
- 마케터 채용 → 전문적 마케팅
- 멘토링 → 사업 전략 구체화

### Q8: Exit 전략은?
**A**: 2가지 옵션이 있습니다.
1. **M&A** (3-5년): 네이버/카카오/쿠팡 인수 (예상 가치 100-300억)
2. **IPO** (5-7년): 코스닥 상장 (연 매출 100억 달성 시)
