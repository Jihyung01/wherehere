### 6. 일괄 전송 (Batch Send)

#### 개선안
```typescript
// 여러 친구에게 한 번에 전송
const sendToMultipleFriends = async (
  friendUuids: string[],
  content: ShareContent
) => {
  const results = await Promise.allSettled(
    friendUuids.map(uuid => sendMessage(uuid, content))
  )
  
  const success = results.filter(r => r.status === 'fulfilled').length
  const failed = results.filter(r => r.status === 'rejected').length
  
  toast.success(`${success}명에게 전송 완료! ${failed > 0 ? `(${failed}명 실패)` : ''}`)
}
```

**UI**:
```typescript
// 친구 목록에서 체크박스로 다중 선택
<FriendList>
  {friends.map(friend => (
    <FriendItem>
      <Checkbox checked={selectedFriends.includes(friend.uuid)} />
      <FriendInfo>{friend.name}</FriendInfo>
    </FriendItem>
  ))}
</FriendList>
<Button onClick={() => sendToMultipleFriends(selectedFriends, content)}>
  선택한 친구에게 전송 ({selectedFriends.length}명)
</Button>
```

**효과**:
- 한 번에 여러 친구에게 전송
- 시간 절약
- 더 강력한 바이럴

### 9. 스마트 추천 (Smart Suggestions)

#### 개선안
```typescript
// AI 기반 공유 대상 추천
const suggestShareTargets = (content: ShareContent) => {
  // 장소 카테고리 기반 추천
  if (content.type === 'place') {
    const category = content.place.category
    return friends.filter(f => 
      f.preferences.includes(category) || 
      f.recentVisits.some(v => v.category === category)
    )
  }
  
  // 최근 대화 기반 추천
  if (content.type === 'quest') {
    return friends
      .filter(f => f.lastChatWithin24h)
      .sort((a, b) => b.chatFrequency - a.chatFrequency)
      .slice(0, 5)
  }
}

// UI
<SuggestedFriends>
  <Title>이 친구들이 좋아할 것 같아요</Title>
  {suggestedFriends.map(friend => (
    <FriendChip onClick={() => selectFriend(friend)}>
      {friend.name}
    </FriendChip>
  ))}
</SuggestedFriends>
```

**효과**:

### 10. 공유 링크 단축 & 추적

#### 개선안
```typescript
// 단축 URL 생성 (UTM 파라미터 포함)
const createShareLink = async (content: ShareContent, userId: string) => {
  const longUrl = `${appUrl}/${content.type}/${content.id}?ref=${userId}`
  
  // 단축 URL 생성 (예: bit.ly, 자체 서비스)
  const shortUrl = await shortenUrl(longUrl, {
    utm_source: 'kakao',
    utm_medium: 'share',
    utm_campaign: content.type,
    utm_content: content.id,
  })
  
  return shortUrl
}

// 클릭 추적
const trackShareClick = async (shortUrl: string) => {
  await fetch('/api/analytics/share-click', {
    method: 'POST',
    body: JSON.stringify({ shortUrl, timestamp: Date.now() })
  })
}
```

**효과**:
- 공유 효과 측정
- 어떤 콘텐츠가 인기 있는지 파악
- 마케팅 데이터 수집
