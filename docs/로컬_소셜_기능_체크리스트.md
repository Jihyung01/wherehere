# 로컬 소셜 기능 체크리스트 (웹페이지 vs 현재 앱)

어제 만든 단일 웹페이지(동네 피드) 기능을 기준으로, 현재 WhereHere 앱에 없는 항목을 정리한 체크리스트입니다.

---

## 1. 작성(Compose) 관련

| 웹페이지 기능 | 현재 앱 상태 | 비고 |
|---------------|-------------|------|
| 글 타입 선택 (동네 이야기 / 리뷰 / 모임) | 없음 | 앱에는 게시글 작성 UI 없음 |
| 제목 + 내용 입력 | 없음 | |
| 리뷰 별점 (1~5) | 퀘스트 리뷰에만 있음 | 로컬 피드용 리뷰 포스트 없음 |
| 모임 시간 입력 (meet_time) | 없음 | |
| 사진 업로드 + 미리보기 + 제거 | 퀘스트 리뷰 사진만 | 로컬 게시글용 이미지 없음 |
| 현재 장소 연결 (place_name, place_address) | 없음 | |
| 작성자명 (카카오 프로필 기준) | 로그인 사용자만 가능 | 친구 피드에는 체크인 활동만 표시 |

---

## 2. 피드(Feed) 관련

| 웹페이지 기능 | 현재 앱 상태 | 비고 |
|---------------|-------------|------|
| 게시글 목록 (시간순) | 없음 | feed_activities(체크인)만 존재 |
| 게시글 타입 라벨 (리뷰/모임/동네 이야기) | 없음 | |
| 댓글 목록 per 게시글 | 없음 | |
| 댓글 작성 (입력 + 등록) | 없음 | |
| 댓글 작성자 + 상대 시간 표시 | 없음 | |
| 게시글별 카카오 공유 버튼 | 없음 | 퀘스트 공유만 있음 |
| 게시글별 인스타 공유 버튼 | 없음 | 링크 복사 수준만 있음 |

---

## 3. 공유(Share) 관련

| 웹페이지 기능 | 현재 앱 상태 | 비고 |
|---------------|-------------|------|
| 카카오 공유 (게시글/장소) | 장소/퀘스트만 텍스트+링크 | 게시글 카드 공유 없음 |
| 인스타 공유: **스토리 사이즈 카드**(1080x1920) 생성 | 없음 | 웹에는 drawStoryCard + canvas |
| 인스타 공유: **피드 사이즈** 카드 | 없음 | |
| 카드 이미지 → Web Share(파일) 또는 다운로드 | 없음 | |
| 인스타용 캡션 복사 (해시태그 포함) | 링크/텍스트만 복사 | 카드+캡션 연동 없음 |

---

## 4. 지표/요약(Metrics) 관련

| 웹페이지 기능 | 현재 앱 상태 | 비고 |
|---------------|-------------|------|
| 동네 이름 표시 (area_title / local_area) | 없음 | 소셜 탭에 "동네" 개념 없음 |
| 오늘 게시글 수 / 열린 모임 수 / 리뷰 평균 / 댓글 수 | 없음 | |
| "동네에서 지금 살아있는 동네 분위기" 요약 문구 | 없음 | |
| 열린 모임 미니 리스트 (상위 3개) | 없음 | |
| 주목받는 리뷰 미니 리스트 (상위 3개) | 없음 | |

---

## 5. UI/탭 구조

| 웹페이지 기능 | 현재 앱 상태 | 비고 |
|---------------|-------------|------|
| 탑 레벨: 홈 / 작성 / 피드 탭 | 없음 | 소셜은 단일 화면(친구찾기+체크인 피드) |
| 동네 vs 친구(팔로우) 피드 분리 | 없음 | 팔로우 기반 피드만 존재 |
| 테마 토글 (라이트/다크) | 있음 | 설정에서 처리 |
| 날짜 표시 (오늘의 장소 영역) | 퀘스트/홈에 있음 | 로컬 피드 헤더에는 없음 |
| 오늘의 장소 패널 (카카오 장소 1곳 추천) | 퀘스트 플로우에 있음 | 로컬 홈과 별개 |

## 7. 수동 검증 가이드

배포 또는 로컬 실행 후 아래 순서로 확인하면 됩니다.

1. **Supabase**: SQL Editor에서 `20260303_local_posts_comments.sql` 실행해 테이블 생성.
2. **소셜 탭 진입**: 앱 하단 네비에서 소셜 선택 → **동네** / **친구** 탭 전환 동작 확인.
3. **동네 작성**: 동네 탭 → 작성 → 글 타입(동네 이야기/리뷰/모임), 제목, 내용 입력 후 "동네 피드에 올리기" → 성공 시 피드 탭에서 해당 글이 보이는지 확인.
4. **댓글**: 동네 피드에서 아무 게시글의 댓글 입력란에 텍스트 입력 후 "댓글 등록" → 댓글이 해당 글 아래에 바로 표시되는지 확인.
5. **친구 피드**: 친구 탭에서 기존처럼 친구 코드 복사, 친구 검색, 팔로우, 체크인 소식 피드가 그대로 동작하는지 확인 (회귀 없음).
6. **인스타 공유**: 동네 피드에서 게시글의 "인스타 카드" 버튼 클릭 → 스토리 크기 카드 이미지가 생성되고, Web Share 지원 시 공유 시트가 뜨거나, 미지원 시 다운로드 + 캡션 복사가 되는지 확인.
