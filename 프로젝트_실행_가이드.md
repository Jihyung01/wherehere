## ⚠️ 주의사항

### 데이터베이스 없이 실행 시
백엔드가 DB에 연결하지 못하면 에러가 발생합니다. 하지만:
- **프론트엔드 UI는 완전히 작동**합니다
- UI 개발/테스트는 프론트엔드만으로 가능합니다
- Mock 데이터를 사용하여 개발할 수 있습니다

### DB가 필요한 경우
```powershell
# Docker로 DB만 실행
docker-compose up -d db

# DB 접속 정보
- Host: localhost
- Port: 5432
- Database: wh_core
- User: wh_user
- Password: wh_password
```

## 🔧 자주 하는 질문

### Q: npm install이 너무 느려요
A: 잠시 기다려주세요. 첫 설치 시 1-2분 정도 소요됩니다.

### Q: 포트 3000이 이미 사용 중이에요
A: 다른 포트로 실행하세요:
```powershell
npm run dev -- -p 3001
```

### Q: 백엔드 에러가 나요
A: DB 연결을 확인하세요. 또는 프론트엔드만 실행하세요.

### Q: Docker가 없어요
A: 백엔드를 로컬에서 실행하거나, 프론트엔드만 실행하세요.
